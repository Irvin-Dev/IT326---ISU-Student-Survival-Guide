# IT326---ISU-Student-Survival-Guide
Principles of Software Engineering Group Project - A forum based software that allows students at ISU to interact with each other regarding classes, along with professors helping students. Project is forum based.


Steps:
-> Download the isu-survival-backend.zip


-> Unzip the file 


-> Connect to ISU VPN or use ISU WiFi to access the database


-> Create a React Project in your IDE of choice


-> Replace the src folder with the extracted src.zip located on the GitHub


-> Run npm start in your terminal to launch the React App


-> In the Spring Boot backend, make sure App Controller is connected to the same port the React application is running on
